# IQS 搜索快速启动指南

## 5分钟快速配置

### 步骤 1: 获取 API Key (2分钟)

1. 访问 [阿里云 IQS 控制台](https://help.aliyun.com/zh/document_detail/3025781.html)
2. 登录阿里云账号（没有则注册）
3. 点击"创建 API Key"
4. 复制生成的 Key（格式类似：`sk-xxxxxxxxxxxxxxxx`）

### 步骤 2: 配置 CoSurf (1分钟)

1. 打开 CoSurf 应用
2. 点击左下角 ⚙️ 设置图标
3. 选择 **工具** 标签页
4. 在"阿里云 IQS 搜索配置"区域粘贴 API Key
5. 点击 **保存** 按钮
6. 看到绿色 ✓ 提示即成功

### 步骤 3: 测试搜索 (2分钟)

在 AI 对话框中输入：

```
帮我搜索今天的科技新闻
```

AI 应该会自动调用搜索工具并返回结果！

## 常见问题

### Q: 为什么需要 API Key？

A: IQS 是阿里云的商业服务，需要 API Key 进行身份验证和计费。

### Q: 收费吗？

A: IQS 按调用次数计费，具体价格请参考阿里云官网。新用户通常有免费额度。

### Q: API Key 安全吗？

A: API Key 存储在本地数据库中，不会上传到云端（除 IQS API 调用外）。建议定期轮换。

### Q: 搜索失败怎么办？

A: 检查：
1. API Key 是否正确配置
2. 网络连接是否正常
3. 查看错误提示信息

### Q: 可以切换搜索引擎吗？

A: 当前仅支持阿里云 IQS。未来版本可能会支持更多引擎。

## 故障排除速查

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| "未配置 API Key" | 未在设置中配置 | 打开设置 → 工具 → 配置 API Key |
| "API 请求失败 (401)" | API Key 无效 | 检查 Key 是否正确，重新获取 |
| "API 请求失败 (429)" | 超出配额 | 等待或升级套餐 |
| "无搜索结果" | 查询词不当 | 尝试更通用或更精确的词 |
| 搜索很慢 | 网络问题 | 检查网络连接 |
